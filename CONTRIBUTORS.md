## Contributing
Pull requests welcome. If you extend support for additional ProLine endpoints, new integrations (CallRail, CompanyCam, QuickBooks, Google Calendar), or add verified workflow recipes, please include a test log entry in `tests/test-log.md` showing what you tested and what you saw.

For integrations that require ProLine-side configuration (webhooks, workflow triggers), please document the ProLine UI steps alongside the code.
